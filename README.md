# materials

Papers posted here are suitable for signing up for Friday presentations.

## specific presentation ideas
- Dasgupta-Maskin, "On the robustness of majority rule" (proof that Condorcet rules dominate all others in a measure-theoretic sense)
- Eric Maskin, "[Borda's rule and Arrow's independence condition](https://maskin.scholars.harvard.edu/sites/g/files/omnuum10606/files/bordas_rule_and_arrows_independence_condition_jpe_vol133_iss2_0.pdf)" (kind of a follow-up, with a modified IIA)
- Ulle Endriss, strategy for proving Arrow's theorem with SAT-solvers:  Slides 1, Slides 2 posted above; more info and code in [course outline](https://staff.fnwi.uva.nl/u.endriss/teaching/advanced-comsoc-2021/) 
- Samuel Baltz, "Simulating electoral system changes and how voters might respond" (estimates levels of real-world strategic voting)
